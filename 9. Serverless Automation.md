# Objective - To automate administrative workflows

This phase focuses on automating administrative workflows and providing real-time visibility into studio asset uploads without taxing the primary web server.

### 1. The "Observer"
I configured the `is-uploads-1` S3 bucket to act as an event source.
* Action: Any time an architect uploads a new 3D render (s3:ObjectCreated:*), S3 sends a JSON payload to AWS Lambda.
* Benefit: This removes the need for the EC2 server to "poll" the bucket for changes, saving CPU cycles and reducing costs.

--- 

### 2. The "Worker" 
I developed a Python-based Lambda function named `is-upload-notifier`.
* Logic: The function parses the S3 event, identifies the filename, and triggers a notification.
* Serverless Advantage: Since Lambda only runs when a file is uploaded, the studio pays $0.00 for this automation when no work is being done.
