# Ideal Studios Cloud Migration
Ideal Studios is a high-end rendering studio that required a secure, scalable and automated cloud environemt. This project involved migrating their local infrastructure to a 3-tier VPC architecture, implementing global content delivery and setting up event-driven automation for asset mangement

## Key Features
#### Security: Isolated database in a private subnet
#### Performance: Latency reduction via Amazon CloudFront
#### Automation: Serverless S3-to-Lambda notification system
#### Reliability: Automated daily backups.
